# Code Review Prompt

## Objective

Review the implementation against the project specifications.

---

## Read

- requirements.md
- architecture.md
- feature specification
- relevant ADRs

---

## Review

Evaluate:

- Architecture
- Code Quality
- SOLID
- Readability
- Maintainability
- Type Safety
- Error Handling
- Performance
- Security
- Edge Cases

---

## Output

Strengths

Weaknesses

High Priority Issues

Medium Priority Issues

Low Priority Issues

Recommendations

Overall Score (1–10)
