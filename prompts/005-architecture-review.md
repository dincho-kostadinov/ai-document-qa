# Architecture Review Prompt

## Objective

Review the implementation from an architectural perspective.

Ignore formatting and style.

Focus on architecture.

---

## Evaluate

- Layered Architecture
- Separation of Concerns
- Coupling
- Cohesion
- Dependency Direction
- Modularity
- Testability
- Extensibility
- Maintainability

---

## Questions

Does the implementation respect the architecture?

Are responsibilities clearly separated?

Is any module doing too much?

Are dependencies pointing in the correct direction?

Can this feature evolve without major refactoring?

---

## Output

Architecture Strengths

Architecture Weaknesses

Improvement Suggestions

Architectural Score (1–10)
